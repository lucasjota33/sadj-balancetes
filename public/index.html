Cloudinary — Upload de Comprovantes
Imagens (JPG/PNG): armazenadas diretamente. PDFs: convertidos automaticamente para imagem — o link abre no navegador sem baixar. Qualquer pessoa com o link consegue visualizar.
No Dashboard do Cloudinary, abaixo do nome da conta
Settings → Upload → Upload presets → Signing mode: Unsigned
Google Drive — Backup dos Comprovantes
Os comprovantes são enviados também para o Google Drive da pasta que você escolher — independente da conta. Usa o mesmo Apps Script já configurado. Os arquivos ficam organizados por mês e tipo (DESPESAS / RECEITAS) e geram um link de preview direto (abre no Google Drive Viewer).
O mesmo script usado para salvar a planilha
Abra a pasta no Drive e copie o ID da URL após /folders/
💡 Dica: Você pode usar só o Cloudinary, só o Drive, ou os dois juntos. Se ambos estiverem configurados, o arquivo vai para os dois destinos automaticamente.
Guia de Configuração

☁️ Cloudinary (PDFs com preview)

1
Acesse cloudinary.com e crie uma conta gratuita
2
No Dashboard, copie o Cloud Name
3
Vá em Settings → Upload → Upload presets → Add upload preset
Mude Signing mode para Unsigned → nome: sadj_comprovantes → Save
4
Cole Cloud Name e Preset nos campos acima → clique em Testar Cloudinary

📁 Google Drive (pasta da sua escolha)

1
Acesse script.google.com → projeto existente ou novo → cole o Code.gs
2
Abra a pasta de destino no Drive, copie o ID da URL e cole em "ID da pasta raiz"
3
No Apps Script: Implantar → Nova implantação → App da Web
Executar como: Eu · Acesso: Qualquer pessoa → Implantar → copie a URL
4
Cole a URL no campo "URL do Apps Script" acima

⚠️ No Code.gs, substitua COLE_O_ID_DA_PASTA_RAIZ_AQUI pelo ID da sua pasta antes de publicar.
